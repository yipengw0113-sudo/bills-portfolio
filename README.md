# Bill's Portfolio

A beginner-friendly investment portfolio landing page with a live quote lookup and a CFA-style stock analyzer.

## Run locally

1. Create a `.env` file:

```env
TWELVE_DATA_API_KEY=your_twelve_data_api_key_here
```

2. Start the server:

```bash
npm start
```

3. Open:

```text
http://127.0.0.1:3000
```

## Deployment note

Do not upload `.env` to GitHub. Add `TWELVE_DATA_API_KEY` as an environment variable in your hosting platform.
